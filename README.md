# 🌱 AgroCloud Pro

**AgroCloud Pro** is a modern, single-page smart farming dashboard that combines weather intelligence, crop support tools, and AI-powered assistance in one clean interface.

## ✨ What makes it interesting

- 🤖 **Agro AI Assistant** for farming Q&A and guidance  
- 🦠 **Plant disease support flow** with image upload and treatment suggestions  
- 🌦️ **Live weather panel** with location search + GPS support  
- 💧 **Smart irrigation planner** to help optimize watering decisions  
- 🌾 **AI crop advisor** for practical crop recommendations  
- 🎨 **Premium UI feel** with animations, gradients, and responsive design

## 🧩 Tech Snapshot

- **Frontend:** HTML + Tailwind CSS (CDN)
- **Icons:** Lucide
- **Storage:** Browser `localStorage` for profile persistence
- **APIs used in app logic:** Gemini, Open-Meteo, Nominatim, Plant.id

## 🚀 Run locally

Since this project is a standalone static page:

1. Clone the repository
2. Open `/home/runner/work/The-Pro-One/The-Pro-One/index.html` in your browser  
   *(or serve the folder with any static server for best results)*

## 🔐 Configuration note

The `index.html` contains an `apiKey` variable for AI features.  
Set a valid key if you want full Gemini-powered functionality while running locally.
